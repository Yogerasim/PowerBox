# Sorting Notes

Исходная папка UDPRele скопирована в:

Archive/UDPRele-original/

Эта папка является архивом.
Файлы из нее нужно постепенно разобрать и перенести в правильные разделы.

## Куда переносить файлы

Старые прошивки ESP8266 D1:

Firmware/legacy-esp8266-d1/

Новые прошивки ESP32-S3:

Firmware/esp32-s3-main/

Тестовые прошивки ESP32-C3:

Firmware/esp32-c3-tests/

MIDI / Roland T-8 тесты:

Firmware/midi-t8-tests/

Актуальные TouchDesigner-файлы:

TouchDesigner/current/

Старые TouchDesigner-файлы:

TouchDesigner/legacy/

Тестовые TouchDesigner-файлы:

TouchDesigner/tests/

## Важно

Не удалять файлы из Archive/UDPRele-original/, пока они не будут точно перенесены и проверены.
