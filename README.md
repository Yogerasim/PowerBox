# PowerBox

**PowerBox** — универсальная коробка управления нагрузками 220V на 8 розеток.

Проект нужен для инсталляций, света, сценографии, TouchDesigner-сцен, MIDI-контроля и быстрого включения/выключения электрических устройств.

## Что хранится в репозитории

- TouchDesigner-патчи
- прошивки для ESP32
- схемы подключения
- документация по сборке
- инструкции по управлению коробкой
- заметки по безопасности и эксплуатации

## Возможности

PowerBox позволяет управлять 8 каналами нагрузки:

- лампы 220V
- LED-лампы 220V
- блоки питания для низковольтных LED-лент
- световые приборы
- небольшие моторы и вентиляторы
- дым-машины или другие устройства, если они подходят по мощности
- любые безопасные нагрузки в пределах допустимой мощности реле и проводки

## Управление

Управление может происходить через:

- UDP-команды из TouchDesigner
- веб-интерфейс ESP32
- MIDI-сигналы
- локальную Wi-Fi сеть
- fallback AP-режим ESP32, если нет внешней сети

## Базовая архитектура

TouchDesigner / MIDI / Web UI  
↓  
UDP / Wi-Fi / UART MIDI  
↓  
ESP32  
↓  
8-channel relay module  
↓  
8 розеток 220V

## UDP-протокол

Коробка принимает UDP-команды на порт:

4210

Примеры команд:

ON 1  
OFF 1  
TOGGLE 1  
ALL ON  
ALL OFF  
STATUS  
10101010

Где `10101010` — это битовая маска для 8 каналов.

Примеры:

`10000000` — включает первый канал и выключает остальные.  
`11111111` — включает все 8 каналов.  
`00000000` — выключает все 8 каналов.

## Wi-Fi

ESP32 может работать в двух режимах.

### 1. Подключение к существующей Wi-Fi сети

Если в прошивке указаны данные сети, коробка подключается к роутеру и принимает UDP-команды по локальному IP.

### 2. Fallback AP

Если ESP32 не может подключиться к Wi-Fi, она поднимает свою точку доступа:

PowerBox-Setup

IP по умолчанию:

192.168.4.1

## Подключение 220V

Внутри коробки используется 8-канальный релейный модуль. Каждое реле управляет отдельной розеткой.

Примерная логика:

220V input  
↓  
общий ввод фазы  
↓  
relay 1 -> socket 1  
relay 2 -> socket 2  
relay 3 -> socket 3  
relay 4 -> socket 4  
relay 5 -> socket 5  
relay 6 -> socket 6  
relay 7 -> socket 7  
relay 8 -> socket 8  

Ноль и земля разводятся отдельно на розетки.

## Важно по безопасности

PowerBox работает с напряжением 220V. Сборка и эксплуатация должны выполняться аккуратно и с пониманием электробезопасности.

Рекомендации:

- использовать предохранитель на вводе
- использовать IEC C14 с выключателем и fuse
- не превышать ток реле
- не превышать сечение проводов
- разделять силовую и управляющую часть
- не работать с открытой коробкой под напряжением
- проверять изоляцию и заземление
- подписывать все каналы и провода
- использовать кабельные вводы и фиксацию проводов
